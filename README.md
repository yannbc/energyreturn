# energyreturn

Personal website, CV variants, and job application materials for Yann Burden.

## Repo structure

```
site/               Deployed to GitHub Pages (energyreturn.co)
  index.html        Landing page
  me/index.html     Data-driven CV with flavour switcher
  CNAME             Custom domain config

cv/                 CV source files (not deployed)
  cv-base.md        Base CV, data science and AI emphasis
  cv-consulting.md  Consulting and advisory variant

applications/       Cover letters and application materials
  quantium/         Quantium role applications

media/              Transcripts, interviews, press
  tech-trajectory-transcript.txt
```

## Deployment

The `site/` folder is published via GitHub Pages to [energyreturn.co](https://energyreturn.co). The Pages source is configured to serve from the `site/` directory on the `main` branch.
